# 2022 Project Auction App
프로젝트 주제 : 온라인 경매 앱 제작

#### FrontEnd
* Kotlin (Jetpack Compose)

#### Backend
* Firebase (.json파일 이용하여 연결)

#### Tool
* IntelliJ
* Figma

## Module

### data 

게시글, 물품정보등의 정보를 저장하는 object클래스

### function 

정보수정, 정보가져오기등의 백엔드 기능

### layout

앱 전체 화면을 구성하는 패키지

### shapes

각 화면의 컴포넌트들을 구성하는 패키지
